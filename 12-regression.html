13  回归分析

回归分析是统计学核心算法，是计量模型和机器学习的最基本算法。

计量模型和机器学习中的各种回归算法都可以看作是线性回归的扩展，即使是分类算法也可以看做是一种特殊的回归

回归分析是确定两个或两个以上变量相互依赖的定量关系的一种统计分析方法。具体是通过多组自变量和因变量的样本数据，拟合出最佳的函数关系。

回归分析通常用于：

  • 探索现象-结果的影响因素主要有那些？
  • 影响因素对现象/结果是怎样影响的？
  • 预测未来的现象/结果。

13.1 回归建模的数学描述

设y为因变量数据，x为自变量数据，假设两者间的真实（精确）关系为：

y=f(x) y=f(x)

但在实际情况下，回归模型只能试图找到一种近似的关系代替它：

f̂(x)≈f(x) \hat{f}(x) \approx f(x)

两者之差就是模型的残差：

ξ=f(x)−f̂(x) \xi = f(x)-\hat{f}(x)

我们总是希望把y和x的关系都留在模型部分，让残差部分最好只是白噪声（完全是随机误差，0均值，微小标准差的正态分布）：

ξN(0,θξ2) \xi {~} N(0,\theta_\xi^2)

此时说明回归模型建模成功，否则，就是模型尚未提取出充分的模型关系（欠拟合）。

Note

回归模型的基本原则是：在没有显著差异的情况下，优先选择更简单的模型。如果更简单的模型已经足已充分建模，强行使用复杂模型会产生过拟合，反而降低模型的泛化（预测）能力

13.2 回归诊断

13.2.1 拟合优度检验

即 R2R^2，也成为可决系数（决定系数），反映了自变量能解释的方差占总方差的百分比。R2R^2 值越大，说明模型拟合的结果越好。

R2R^2 的计算未考虑自由度的问题，为避免增加自变量而高估 R2R^2，对 R2R^2 进行调整是更合理的：Radj2=1−n−1n−p−1×(1−R2)R^2_{adj}=1-\frac{n-1}{n-p-1} \times (1-R^2), n为样本数，p为自变量个数。

13.2.2 均方误差和均方根误差

均方误差：

MSE=1n∑i=1n(yi−yî)2 MSE = \frac{1}{n}\sum_{i=1}^n(y_i-\hat{y_i})^2

均方根误差：刻画预测值与真实值平均偏离情况，是所有回归模型（包括机器学习中的回归算法）中最常用的性能评估指标。

RMSE=1n∑i=1n(yi−yî)2 RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^n(y_i-\hat{y_i})^2}

13.2.4 共线性回归

多元线性回归建模，若自变量数据之间存在较强的线性相关性，即存在多重共线性。此现象会导致回归模型不稳定，得到的模型被称为伪回归模型，并不能反映自变量与因变量之间的真实关系。

多元线性回归建模，需要做共线性诊断，识别出多重共线性，并处理多重共线性再建模。

多重共线性的解决办法（任选其一）：

  • 若两个自变量线性相关系数较大，则只用其中一个。
  • 使用逐步回归，剔除冗余自变量。
  • 使用主成分回归，相当于对自变量进行重组，在做回归。
  • 使用正则化回归：岭回归、Lasso回归、弹性网模型（前两组合）。

13.3.2 构建多元线性回归模型

lm(formula, data, …)：拟合多元线性回归模型。

  • formula设定模型公式，遵从Wilkinson表示规则，这里列举一些常用的写法：

    • y ~ .：包含所有自变量。
    • x1 : x2：交互效应，即 x1x2x_1x_2 项。
    • x1 * x2：包括主效应和交互效应，即 x1+x2+x1x2x_1+x_2+x_1x_2 的简写。
    • I()：打包式子作为整体。
    • y ~ ploy(x, 2, raw = TRUE)： 一元二次多项式回归，同 y ~ x + I(x^2)。
    • y ~ ploym(x1, x2, degree = 2, raw = TRUE)： 二元二次多项式回归。
    • log(y) ~ x： 对y做对数变换。

这里我们先将所有自变量都用上，构建初始的多元线性回归模型（往往不是成功的模型）

mdl0 <- lm(body_mass ~ ., penguins)

13.3.4 回归模型中的分类变量

分类变量，即取值是有限的类别值，如性别的取值为有限的男、女。分类变量不能直接应用到回归模型中，即使用0和1分别表示男女也不行，因为如果不加处理，计算机会将其单纯作为数值型变量处理。

R中，将分类变量转换为因子型或数值型，就可以不需要进行进一步处理。

13.3.5 模型改进

自变量又称为特征，利用原有自变量构造新的自变量，就是特征工程。特征工程是改进模型的重要手段，也是数据挖掘/机器学习中的关键步骤。

多元线性回归相当于是用 1 次多项式去逼近真实的函数关系，如果提高的 2次，即把所有二次项包括交互项：x12,x22,x1x2,...x_1^2,x_2^2,x_1x_2,...都加入模型，拟合效果大 概率会有提升。但新加入的项，可能会有不显著或产生共线性。解决办法，就是用逐步回归进行变量筛选。

关于交互项x1x2x_1x_2的解释：x1对y的影响受x2的调节，反之亦然。

常用的构建特征方法还有：对特征做各种变换，连续特征离散化，比如年龄相差 1 岁影响不一定显著，但年龄段的差异，比如从青年到中年到老年，很可能会显著。

本案例中，我们将三个数值变量的二次项，以及交互项sex:island加入模型，在逐步回归剔除不显著项。

mdl2 <- lm(body_mass ~ species 
  + sex * island + bill_length 
  + I(bill_length^2) + bill_depth 
  + I(bill_depth^2) + flipper_length
  + I(flipper_length^2), penguins) %>% 
  step(direction = "backward", trace = 0)
summary(mdl2)

13.3.6 回归诊断

  • 残差检验

理想的模型（标准化）残差应服从“0 均值小方差”（标准）正态分布，对于残差，通常是绘制（标准化）残差图、残差 QQ 图、残差直方图，或者对（标准化）残差的正态性、独立性、异方差性做统计检验。

  • 强影响分析 对参数估计或预测值有异常影响的数据，称为强影响数据。回归模型应当具有一定的稳定性，若个别样本数据对估计有异常大的影响，剔除后将得到与原来差异很大的回归方程，从而有理由怀疑原回归方程是否真正描述了变量间的客观存在的关系。这些强影响样本是异常值，应当识别出来剔除之后，再重新拟合回归模型。
使用ggfortify::autoplot()绘制回归诊断图，包括残差图、残差 QQ图、标准化残差图、强影响图等，还能同时标记强影响样本。
library(ggfortify)

## 回归诊断

### 拟合优度检验

即 $R^2$，也成为可决系数（决定系数），反映了自变量能解释的方差占总方差的百分比。$R^2$ 值越大，说明模型拟合的结果越好。[$R^2$ 的计算未考虑自由度的问题，为避免增加自变量而高估 $R^2$，对 $R^2$ 进行调整是更合理的：$R^2_{adj}=1-\frac{n-1}{n-p-1} \times (1-R^2)$, n为样本数，p为自变量个数。]{.aside}

### 均方误差和均方根误差

均方误差：

$$
MSE = \frac{1}{n}\sum_{i=1}^n(y_i-\hat{y_i})^2
$$

均方根误差：刻画预测值与真实值平均偏离情况，是所有回归模型（包括机器学习中的回归算法）中最常用的性能评估指标。

$$
RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^n(y_i-\hat{y_i})^2}
$$

### 共线性回归

多元线性回归建模，若自变量数据之间存在较强的线性相关性，即存在**多重共线性**。此现象会导致回归模型不稳定，得到的模型被称为*伪回归模型*，并不能反映自变量与因变量之间的真实关系。

多元线性回归建模，需要做**共线性诊断**，识别出多重共线性，并处理多重共线性再建模。

多重共线性的解决办法（任选其一）：

- 若两个自变量线性相关系数较大，则只用其中一个。
- 使用逐步回归，剔除冗余自变量。
- 使用主成分回归，相当于对自变量进行重组，在做回归。
- 使用正则化回归：岭回归、Lasso回归、弹性网模型（前两组合）。

### 构建多元线性回归模型

lm(formula, data, ...)：拟合多元线性回归模型。

- formula设定模型公式，遵从Wilkinson表示规则，这里列举一些常用的写法：

   - `y ~ .`：包含所有自变量。
   - `x1 : x2`：交互效应，即 $x_1x_2$ 项。
   - `x1 * x2`：包括主效应和交互效应，即 $x_1+x_2+x_1x_2$ 的简写。
   - `I()`：打包式子作为整体。
   - `y ~ ploy(x, 2, raw = TRUE)`： 一元二次多项式回归，同 y ~ x + I(x^2)。
   - `y ~ ploym(x1, x2, degree = 2, raw = TRUE)`： 二元二次多项式回归。
   - `log(y) ~ x`： 对y做对数变换。

这里我们先将所有自变量都用上，构建初始的多元线性回归模型（往往不是成功的模型）

```{r}
mdl0 <- lm(body_mass ~ ., penguins)
```

### 共线性针对与逐步回归

1. 使用`car::vif()`诊断回归模型的多重共线性：[mctest::imcdiag() 诊断回归模型的多重共线性更全面，除了计算 VIF 值外，还计算其他诊断指标值。]{.aside}

```{r}
car::vif(mdl0)
```

只有分类变量的species的VIF值较大，其余均小于10，说明不存在共线性。

2. 使用`step()`函数做逐步回归，剔除不显著的和共线性的自变量。逐步回归是以 AIC 值（越小越好）作为加入和剔除变量的判别条件，参数direction 设置逐步选择的方法： “both”, “backward”（逐步剔除）,“forward”（逐步加入）。[Akaike 信息准则（AIC）常用来比较不同回归模型的拟合效果，优点是既考虑模型的拟合效果又对模型参数过多施加一定惩罚。]{.aside}

```{r}
mdl1 <- step(mdl0, direction = "backward", trace = 0)
summary(mdl1)
```

结果给出了回归系数的标准误、显著性、回归模型的标准误等信息，基于理论的回归系数的置信区间，可以使用`confint()`提取。

```{r}
confint(mdl1)
```

由结果看出，该模型基本拟合成功，回归系数是显著的，模型的调整 $R_2$ 为0.873。

```{r}
# 计算模型均方根误差
library(modelr)
rmse(mdl1, penguins)
```
### 回归模型中的分类变量

分类变量，即取值是有限的类别值，如性别的取值为有限的男、女。分类变量不能直接应用到回归模型中，即使用0和1分别表示男女也不行，因为如果不加处理，计算机会将其单纯作为数值型变量处理。

R中，将分类变量转换为因子型或数值型，就可以不需要进行进一步处理。

### 模型改进

自变量又称为特征，利用原有自变量构造新的自变量，就是特征工程。特征工程是改进模型的重要手段，也是数据挖掘/机器学习中的关键步骤。

多元线性回归相当于是用 1 次多项式去逼近真实的函数关系，如果提高的 2次，即把所有二次项包括交互项：$x_1^2,x_2^2,x_1x_2,...$都加入模型，拟合效果大
概率会有提升。但新加入的项，可能会有不显著或产生共线性。解决办法，就是用逐步回归进行变量筛选。[关于交互项$x_1x_2$的解释：x1对y的影响受x2的调节，反之亦然。]{.aside}

常用的构建特征方法还有：对特征做各种变换，连续特征离散化，比如年龄相差 1 岁影响不一定显著，但年龄段的差异，比如从青年到中年到老年，很可能会显著。

本案例中，我们将三个数值变量的二次项，以及交互项sex:island加入模型，在逐步回归剔除不显著项。

```{r}
mdl2 <- lm(body_mass ~ species 
  + sex * island + bill_length 
  + I(bill_length^2) + bill_depth 
  + I(bill_depth^2) + flipper_length
  + I(flipper_length^2), penguins) %>% 
  step(direction = "backward", trace = 0)
summary(mdl2)
```

模型mdl2的修正$R_2$比mdl1的稍有提高，说明mdl2比mdl1有所改进，但同时也增加了模型的复杂程度。那么那个模型更好呢？[这里我们遵循一个基本原则，即在模型没有显著差异的情况下，优先选择更简单的模型]{.aside}

可以使用似然比检验`lmtest::lrtest()`或方差分析`anova()`比较两个模型是否有显著差异。

```{r}
anova(mdl1, mdl2)
```

可以看出，检验结果显示检验P值=0.025（小于0.05），说明两个模型有显著差异，应该选择mdl2。

### 回归诊断

- **残差检验**

理想的模型（标准化）残差应服从“0 均值小方差”（标准）正态分布，对于残差，通常是绘制（标准化）残差图、残差 QQ 图、残差直方图，或者对（标准化）残差的正态性、独立性、异方差性做统计检验。

- **强影响分析**
对参数估计或预测值有异常影响的数据，称为强影响数据。回归模型应当具有一定的稳定性，若个别样本数据对估计有异常大的影响，剔除后将得到与原来差异很大的回归方程，从而有理由怀疑原回归方程是否真正描述了变量间的客观存在的关系。这些强影响样本是异常值，应当识别出来剔除之后，再重新拟合回归模型。[使用ggfortify::autoplot()绘制回归诊断图，包括残差图、残差 QQ图、标准化残差图、强影响图等，还能同时标记强影响样本。]{.aside}

```{r}
library(ggfortify)
# 6个可选图形
autoplot(mdl2, which = c(1:3, 6))

# 残差正态性检验
shapiro.test(mdl2$residuals)

# 残差独立性检验
library(lmtest)
dwtest(mdl2)

# 残差异方差检验
bptest(mdl2)
```

通过以上结果，可见mdl2可以通过残差正态性、独立性检验。
