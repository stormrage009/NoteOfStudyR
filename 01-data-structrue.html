1  数据结构

  1. [[]]和[]：

  • [[]]始终提取一个元素的内容，如提取一个列表某一个成分的内容，结果返回为元素（下一级元素）。

  • []始终提取子集，如提取列表的子集，返回的结果还是列表。

1.1 列表

  1. purrr包中提供了一些系列操作列表的函数：

  • pluck()：同 “[[” 提取列表中的元素。

  • keep()： 保留满足条件的元素。

  • discard()： 删除满足条件的元素。

  • compact()： 删除列表中的空元素。

  • append()：在列表末尾增加元素。

  • flatten()： 摊平列表（只摊平一层）。

1.2 因子

R 提供了因子这一数据结构（容器），专门用来存放名义型和有序型的分类变量。因子本质上是一个带有水平（level）属性的整数向量，其中 “水平” 是指事前确定可能取值的有限集合。例如，性别有两个水平：男、女。

forcats包是处理因子的强大工具。

1.3 字符串

字符串是用双引号或单引号括起来的若干字符1，字符串构成的向量，称为字符向量。数据清洗、可视化等操作都会用到字符串处理。

tidyverse 系列中的 stringr包 包提供了一系列接口一致的、简单易用的字符串操作函数，足以代替 R 自带字符串函数2。

两点说明：

  • 查找匹配的各个函数，只是查找第一个匹配，要想查找所有匹配，各个函 数都有后缀 _all 版本；
  • 各个函数中的参数 pattern 都支持用正则表达式。

1.4 日期时间

日期时间通常以字符串的形式传入R中，然后转化为以数值形式储存的日期时间变量。

R的内部日期是以1970年1月1日1至今的天数来储存，内部时间则是以1970年1月1日至今的秒数来储存的。

lubridata包提供了一系列更方便的函数用来生成、管理日期。

Note

不要直接按字符串处理日期时间，应该先转换为日期对象后在处理。

1.4.3 提取日期时间数据的组建

t <- ymd_hms("2022-04-22 17:41:45")
# 提取年份，类似的还有month()、day()、hour()、minute()、second()
year(t)
[1] 2022
# 提取季度
quarter(t)
[1] 2
# 计算时间是当年的第几天
yday(t)
[1] 112
# 关于星期
weekdays(t)
[1] "星期五"
wday(t)  # 日期是本周的第几天，注意默认周日为第1天
[1] 6
wday(t, label = TRUE)
[1] 周五
Levels: 周日 < 周一 < 周二 < 周三 < 周四 < 周五 < 周六
week(t)  # 本周是当年的第几周
[1] 16
Note
  1. 使用**_tz()函数修改时区：
  • tz()显示当前时区
  • with_tz()将时间数据转换为另一个时区的同一时间
  • force_tz()将时间数据的时区强制转换为另一个时区
tz(t)
[1] "UTC"
with_tz(t, tz = "America/New_York")
[1] "2022-04-22 13:41:45 EDT"
force_tz(t, tz = "America/New_York")
[1] "2022-04-22 17:41:45 EDT"
  1. 模糊提取到不同的时间单位，以下函数均需设定取整单位，默认为second
  • 使用round_date()四舍五入时间
  • 使用floor_date()和ceiling_date()向上或向下取整时间
round_date(t, unit = "hour")  # 按小时四舍五入取整
[1] "2022-04-22 18:00:00 UTC"
floor_date(t, unit = "hour")
[1] "2022-04-22 17:00:00 UTC"
ceiling_date(t, unit = "hour")
[1] "2022-04-22 18:00:00 UTC"

1.4.4 时间段数据

  1. interval()函数计算两个时间点的时间间隔，返回时间段数据。通常需要对时间段进行进一步识别。
begin <- ymd_hm("2019-08-10, 14:00")
end <- ymd_hm("2022-4-26, 9:10")
gap <- interval(begin, end)
gap  # 返回时间段数据
[1] 2019-08-10 14:00:00 UTC--2022-04-26 09:10:00 UTC
# 计算时间段有几天，同理可以计算有几个月、几分钟等
time_length(gap, "day")
[1] 989.7986
  1. duration()函数和period()函数计算数值+时间单位存储时段的长度。 period()函数考虑到了闰年和闰秒，所以在国内更推荐使用。
period(100, units = "seconds")  # 100天有多少秒、共几个星期
[1] "100S"
period(c(90, 5), c("second", "minute"))
[1] "5M 90S"

# 数据结构 {#data-structrue}

1.  `[[]]`和`[]`：

-   `[[]]`始终提取一个元素的内容，如提取一个列表某一个成分的内容，结果返回为元素（下一级元素）。

-   `[]`始终提取子集，如提取列表的子集，返回的结果还是列表。

## 列表 {#sec:list}

1.  `purrr`包中提供了一些系列操作列表的函数：

-   `pluck()`：同 "[[" 提取列表中的元素。

-   `keep()`： 保留满足条件的元素。

-   `discard()`： 删除满足条件的元素。

-   `compact()`： 删除列表中的空元素。

-   `append()`：在列表末尾增加元素。

-   `flatten()`： 摊平列表（只摊平一层）。

```{r}
library(tidyverse)
a <- list(A = 1, B = c(TRUE, FALSE), C = c("a", "b", "c"))
pluck(a$A)

flatten(a)
```

## 因子 {#sec:factor}

R 提供了因子这一数据结构（容器），专门用来存放名义型和有序型的分类变量。因子本质上是一个带有水平（level）属性的整数向量，其中 "水平" 是指事前确定可能取值的有限集合。例如，性别有两个水平：男、女。

[forcats包](https://forcats.tidyverse.org/)是处理因子的强大工具。

## 字符串 {#sec:str}

字符串是用双引号或单引号括起来的若干字符1，字符串构成的向量，称为字符向量。数据清洗、可视化等操作都会用到字符串处理。

tidyverse 系列中的 [stringr包](https://stringr.tidyverse.org/) 包提供了一系列接口一致的、简单易用的字符串操作函数，足以代替 R 自带字符串函数2。

两点说明：

## 日期时间 {#sec:date}

日期时间通常以字符串的形式传入R中，然后转化为以数值形式储存的日期时间变量。

R的内部日期是以1970年1月1日[^data-structrue-1]至今的天数来储存，内部时间则是以1970年1月1日至今的秒数来储存的。

[^data-structrue-1]: 可以参考[这篇文章](https://zhuanlan.zhihu.com/p/29748774)了解有趣最初时间点设置。

[lubridata包](https://lubridate.tidyverse.org/)提供了一系列更方便的函数用来生成、管理日期。

::: {.callout-note}
不要直接按字符串处理日期时间，应该先转换为日期对象后在处理。
:::
